# GitHub Resume Portfolio

A professional portfolio website showcasing projects, experience, education, skills, publications, and personal interests.

## Project Structure

```
Github Resume/
├── .github/              # GitHub Pages configuration
├── assets/               # All static assets
│   ├── css/             # Stylesheets
│   │   └── styles.css
│   ├── js/              # JavaScript files
│   │   └── script.js
│   ├── images/          # Image assets
│   │   ├── personal/    # Personal/travel photos (Scuba.jpeg, Climbing.jpg, Hiking.jpeg, etc.)
│   │   └── projects/    # Project screenshots and GIFs
│   │       ├── NLT_Digital/
│   │       ├── Desk positioning system/
│   │       ├── WMORE/
│   │       ├── VR_rehabilitation assessment tool/
│   │       └── multimodal movement study/
│   └── documents/       # Downloadable files
│       └── CV_Sami-KAAB.pdf
├── index.html           # Main HTML file
├── links.txt            # Reference links
└── .gitignore          # Git ignore rules
```

## Features

- **Responsive Design**: Mobile-friendly layout
- **Interactive Projects**: Modal dialogs with detailed project information
- **Travel Map**: Interactive Leaflet.js map showing countries visited/lived in
- **Social Links**: LinkedIn, GitHub, ORCID integration
- **Downloadable CV**: Professional PDF resume

## Technologies

- HTML5, CSS3, JavaScript
- Leaflet.js 1.9.4 (interactive maps)
- Font Awesome 6.4.0 (icons)
- GeoJSON (country boundary data)

## Sections

1. **Home**: Introduction and call-to-action
2. **About**: Professional summary
3. **Projects**: Featured projects including NLT Digital medical devices, desk positioning system, WMORE IMU platform, VR rehabilitation tools
4. **Experience**: Work history at NLT Digital, Gelomics, University of Queensland
5. **Education**: Academic background
6. **Skills**: Technical competencies in embedded systems, biomedical engineering, IoT
7. **Publications**: Research papers and contributions
8. **Beyond Work**: Personal interests including scuba diving (Level 2 ANMP), climbing, hiking, travel map with 20+ countries
9. **Contact**: Contact information
